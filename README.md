# calismio

Öğrenci paneli (Çalışmıyo) — Vite, React 19, TypeScript, Tailwind CSS.

## Geliştirme

```bash
npm install
npm start
```

Tarayıcı: http://127.0.0.1:5173/

## Üretim build

```bash
npm run build
```

Çıktı: `dist/` — Vercel vb. için uygun.

## Maskot (PNG şeffaf zemin)

```bash
npm run mascot:transparent
```
